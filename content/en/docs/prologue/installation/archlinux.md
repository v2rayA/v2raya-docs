## Install V2Ray core / Xray core

Install `v2ray` from Arch Linux official software source, or install `xray` or `xray-bin` from the AUR.

## Install v2rayA

Install `v2raya` or `v2raya-bin` &amp; `v2raya-git` from the AUR.
